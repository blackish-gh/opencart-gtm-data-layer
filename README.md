# opencart-gtm-data-layer

Google Tag Manager E-commerce Tracking for Opencart 2.x Ocmod
<br><br>
This prints Data Layer when customers make orders, visit the product and category pages or add the product to cart on product page.
<br><br>
Installation. As a usual module. Just upload it to modification installer on admin page or install.xml(better to rename) to "system" folder, then refresh the modifications.
<br><br>
Also you can use this instructions below to make other features work.
<br><br>
To track the click on product read the "click on product.txt"
<br><br>
To track the add to cart or remove from cart actions read the "cart actions.txt"
<br><br>

Google Tag Manager E-commerce Tracking для Opencart 2.x Ocmod
<br><br>
Этот модуль выводит Data Layer на странице успешного заказа, при просмотре страницы товара, категорий или когда товар добавляется в корзину с карточки товара.
<br><br>
Установка. Как обычно, установка через админку или при загрузке install.xml(лучше переименовать) в папку "system" и обязательном обновлении модификаторов.
<br><br>
Также можете воспользоваться другими возможностями отслеживания с помощью дополнительных инструкций.
<br><br>
Чтобы отслеживать клик по товару, прочтите "click on product.txt"
<br><br>
Чтобы отслеживать добавление и удаление товара из корзины, прочтите "cart actions.txt"
